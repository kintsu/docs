### Small example

Here is a compact example to show Kintsu's flavor. The site includes syntax highlighting for `kintsu` sources.

```kintsu
namespace demo;

struct User {
	id: i64,
	name: str,
	email: str,
};

error UserError {
  NotFound { id: i64 }
};

#[err(UserError)]
operation get_user(id: i64) -> User!;
```

### Why Kintsu?

- Concise syntax for types and operations.
- Strong documentation and specification-driven development.
- Designed for readable RFCs and machine-friendly compilation.
