title: "Naming Conventions"
---

| Identifier Of  | Case       |
| -------------- | ---------- |
| enum           | PascalCase |
| enum variant   | PascalCase |
| struct         | PascalCase |
| struct field   | snake_case |
| error          | PascalCase |
| error variant  | PascalCase |
| type alias     | PascalCase |
| operation      | snake_case |
| operation args | snake_case |
